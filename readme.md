Timo — это простое десктоп-приложение для учёта задач и времени, затраченного на них.  
Работает локально, без интернета и без регистрации.

Приложение создано для личного использования и постепенно развивается.

---

## ✨ Возможности

- 📋 Создание задач
- ⏱️ Учёт времени по каждой задаче
- 🗑️ Мягкое удаление задач (данные не теряются)
- 💾 Все данные хранятся локально (SQLite)
- 🚀 Работает без установки Go или базы данных

---

## 🖥️ Как запустить приложение

### macOS

1. Скачайте файл `timo`
2. Откройте папку с файлом
3. Нажмите **правой кнопкой мыши → Открыть**
4. Подтвердите запуск приложения

> ⚠️ macOS может показать предупреждение о безопасности — это нормально для приложений без подписи.

---

## 💾 Где хранятся данные

Все данные сохраняются **на вашем компьютере** в SQLite-базе.
